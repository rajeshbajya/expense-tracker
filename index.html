🚀 Quick Setup Required (One-time only)

To sync your expenses with Google Sheets, follow these steps:

  1. Create a new Google Sheet in your Google Drive
  2. Add headers: ID | Date | Amount | Category | Subcategory | Description
  3. Copy the Sheet ID from the URL (the long string between '/d/' and '/edit')
  4. Make the sheet public: Share → Anyone with the link can view
  5. Paste the Sheet ID below and click "Connect"
